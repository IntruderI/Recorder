⚠️ Важное ограничение браузера:

API транскрибации (SpeechRecognition) всегда слушает устройство записи по умолчанию в Windows/macOS. Он игнорирует внутренние потоки вкладки.

✅ Чтобы транскрибировать вкладку:

  1. Установите Stereo Mix или VB-Cable как "Устройство записи по умолчанию" в системе.
  2. Направьте звук браузера на этот виртуальный кабель.
  3. Вернитесь сюда и нажмите "Проверить вход".
